# 🌲 Random Forest Classification - Iris Dataset 

## 📌 Project Overview

This project focuses on building a Random Forest classifier to improve prediction accuracy and reduce overfitting compared to a single decision tree.

---

## 🧠 Objective

* Train an ensemble model using multiple decision trees
* Improve classification accuracy
* Analyze feature importance

---

## 📊 Dataset

* Iris Dataset
* Features:

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width
* Target:

  * Species (setosa, versicolor, virginica)

---

## 🤖 Model Used

* Random Forest Classifier (Scikit-learn)

---

## 📈 Model Evaluation

* Accuracy Score
* Precision
* Recall
* F1-Score

### ✅ Result

* Achieved high accuracy (~95%–100%)
* Improved performance compared to a single Decision Tree

---

## 🌳 Feature Importance

* Identified which features contributed most to predictions
* Helped understand model behavior

---

## 🧠 Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib
